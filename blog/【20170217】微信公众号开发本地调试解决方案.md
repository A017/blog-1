### 1. 解决方案
内网端口映射到外网访问，然后在微信公众号的外网地址，设置成 Ngrok 生成的地址。

这样本地就可以调试，大大加速了开发的效率和进度。

### 2. Ngrok 的使用
1. 下载Ngrok
2. 注册并生成密钥
3. 运行
```bash
# 把18080 端口映射到外网
ngrok http 18080
```   
>生成的万网地址可能访问速度慢了点，但是是可以映射出来的。 在刚开始使用的时候， 除了80端口映射出去了，其他端口映射出去都访问不了， 后面莫名其妙就可以了。 可能是访问太慢的问题。
